# Schoolwork Assistant
## About
Schoolwork assistant is an Android application that is designed to help students manage notes, and schedule exams, tasks or assessment test centrally.
## Features:
1. Registration and Login
2. 5 Units
3. 3 readable .pdf files which serve as coursework notes for each topic.
4. Adding an activity or event to Google Calendar
5. Go Premium
## Technologies:
1. JDK 21
2. Gradle Version 8.0
3. Compile SDK Version 34 (API 34)
4. Android PDFViewer Library 2.8.2
5. SQLite Database
6. Android Gradle Plugin Version 8.1.3

## Contributing to this project
1. **Fork the Repository**
2. **Clone the Forked Repository**:
   ```bash
   git clone https://github.com/your-username/Schoolwork-Assistant.git
   ```
   Replace `your-username` with your GitHub username.
3. **Create a New Branch**: Navigate to the directory of the cloned repository and create a new branch using the `checkout` command:
   ```bash
   cd Schoolwork-Assistant
   git checkout -b your-feature
   ```
   Replace `your-feature` with the name of your new feature e.g'schedule-an-activity'.
4. **Make Changes**: Make your changes in the new branch.
5. **Commit Your Changes**:
   ```bash
   git add .
   git commit -m "Your commit message"
   ```
   Replace `Your commit message` with your actual commit message.
6. **Push Your Changes**: Push your changes to your forked repository on GitHub:
   ```bash
   git push origin your-feature
   ```
   Replace `your-new-branch` with the name of your new branch.
7. **Create a Pull Request**:
   Go to your forked repository on GitHub, switch to your new branch, and click the 'New pull request' button. Fill in the necessary details and submit the pull request.
